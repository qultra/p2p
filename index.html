🔏 인스타 DM 자동 암호화

비밀번호 하나만 공유하세요.
북마크 한 번으로 보내는 모든 메시지가 자동으로 암호화되고,
받은 암호문은 자동으로 복호화됩니다.

✅ 북마크릿 코드

아래 코드를 복사하여 즐겨찾기(북마크) URL로 등록하세요.

📖 사용 방법

  1. 상대와 똑같은 비밀번호를 약속합니다.
  2. 위 버튼으로 북마크릿 코드를 생성하고 북마크로 저장합니다.
  3. 인스타그램 웹에서 DM 대화를 열고 북마크를 클릭합니다.
  4. 비밀번호를 입력하면, 대화창 옆에 🔒 전송 버튼이 생깁니다.
  5. 평문을 쓰고 🔒 전송을 누르면 암호문이 자동 전송됩니다.
  6. 상대의 암호문은 자동으로 평문으로 보여집니다.
